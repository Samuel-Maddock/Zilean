About Zilean

Zilean is a discord bot that displays live game data and other statistics

Features

Zilean is currently still in development but already has quite a few features. The commands for Zilean are split into various plugins based on there function. A rough list of features is shown below.


Zilean can do the following:

  • Display and search for a LoL Champion and Item
  • Display a match from a summoners match history
  • Display live game information for a summoner that is in a game
  • Add summoners to a tracker list, that automatically displays whether they are in-game every 10 minutes
  • Display various graphs such as kill participation, champion wins, game per month

For more information see the Commands Section!

Development

You can view Zileans GitHub Repo here


Zilean was developed in Python using the following libraries:

  • Discord API Python Wrapper: disco.py
  • League of Legends API Python Wrapper: RiotWatcher
