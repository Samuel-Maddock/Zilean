Patch Notes

Important/Breaking Changes

Version 1.0.7 - Zilean Bot Patch Notes (26/08/18)

Two New Commands:
~changelog - Displays any recent changes to Zilean and links to this page
~iam [summoner_name] [region] - Binds a summoner to your discord account, so you no longer have to type your summoner name in commands

Bug fixes and two new commands:
  • Added a changelog command that displays the latest patch info, and links to the changelog you are currently reading!
  • Added an "I am" command that allows you to bind a summoner to yourself on discord so you no longer have to ener it when typing league commands.
  • Added the changelog page to the site which shows patch notes and important/breaking changes
  • Added a clearer error message when entering the wrong region and the wrong order of arguments
  • Updated site command images to their new updated versions
  • Fixed an issue where Zilean restart messages would break if it didn't have the correct permissions

Version 1.0.6 - Zilean Bot Patch Notes (23/08/18)

Hotfix for:
  • An issue where the you couldn't remove a summoner with a space in their name from the tracker
  • An issue where an error message for if you entered an incorrect region for any league command failed to display fully

Important Zilean Changes From 1.0.5 Onwards

As of adding the default region command in v1.0.5, the command structure for some commands has changed!

For many LoL commands you used to state [region] [summoner_name] but now the order is [summoner_name] (region)
The region is only optional if you set a default region for your server using ~region [league region]

Note that you need to remove any spaces from a summoner name for the commands to work!

Version 1.0.5 - Zilean Bot Patch Notes (22/08/18)

A New Command:
~region [region] - Sets the default region for all league commands for your discord server. By default if you omit the region argument from any command the default region will be used.

Lots of bug fixes and a complete rehaul of how the regions work:
  • Fixed the champion info cmd - Champions with long spell descriptions will now successfully be displayed
  • Disabled the tracker auto cmd for the time being due to performance issues
  • Added the new nexus blitz game mode to Zilean, so it should now display for all match history/live game commands
  • Added a region command which allows a server to use a default region for all league commands
  • The Game Info, Graph and Game Tracker commands will now use the default region if available
  • Added a new patch changes message on the help/info commands that displays some of these changelogs
